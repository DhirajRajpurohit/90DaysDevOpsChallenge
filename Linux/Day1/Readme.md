# Day 3 – Linux Basics
 
Part of my 90 Days DevOps Zero to Hero challenge.
 
## What I Did Today
 
- Learned why Linux is used in DevOps, its features, and basic architecture (kernel, shell, applications).
- Practiced core file system commands: `echo`, `pwd`, `mkdir`, `ls`, `touch`, `cd`, `mv`, `rm`, `rm -d`/`rmdir` vs `rm -rf`, `cp`, `find`, `du -sh`, `df -h`.
- Learned process management commands: `ps aux`, `top`, `htop`, `pidof`, `kill`, `kill -9`, `uptime`, `free -h`.
- Learned the difference between `vim` and `nano` text editors.
- Understood the difference between `apt update` and `apt upgrade`.
- Learned what `sudo` is, created a new user (`dhiraj`), set a password, and switched users with `su`.
- Installed and deployed my first web page using **Nginx**.
